office全家桶

参考链接

安装office全家桶＋visio，此处安装的都是2019专业增强版（建议先安装visio再安装office）。

  1. 安装visio

    1. 下载安装包Visio 2019.zip链接

      提取码: iah4

    2. 解压文件

    3. 打开解压后的文件，右键setup.exe以管理员身份运行

    4. 安装完成，点击关闭

  2. 安装office

    1. 下载office安装包链接
    2. 解压文件
    3. 打开解压后的文件，右键setup.exe以管理员身份运行
    4. 安装完成，点击关闭

  3. 使用MAS工具激活

    1. 下载MAS激活工具（git地址）
    2. 打开文件夹，找到MAS_AIO.cmd；右键以管理员身份运行
    3. 脚本界面见下图。激活方式2、3、5均可激活office，（三种方法任选其一，优先推荐 2=>3=>5 ）
    4. 输入数字2（按键盘数字2）后，进入这个界面
    5. 再按数字1，开始激活
    6. 等待5-10秒，出现绿色框内的文字“office is permanently activated”（office已永久激活），激活成功。点击右上角x关闭此脚本

  4. 打开visio和word或者excel查看，产品已激活！
